Module substrait::types

source ·
Expand description

Methods that perform conversion between Substrait’s type (Type) and GreptimeDB’s type ([ConcreteDataType]).

Substrait use type variation to express different “logical types”. Current we only have variations on integer types. Variation 0 (system preferred) are the same with base types, which are signed integer (i.e. I8 -> i8), and Variation 1 stands for unsigned integer (i.e. I8 -> u8).
